JSON Mobile UI Menu
=============================

Create a mobile UI menu from JSON using [jQuery](http://jquery.com/) and [dust.js](https://github.com/linkedin/dustjs).

The `css` directory stores the [LESS powered](http://lesscss.org/) OOCSS.
